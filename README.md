# Spot Orgánico Backend

Backend y API REST para la plataforma Spot Orgánico.

---

## Funcionalidades

- API REST
- Autenticación JWT
- Gestión de reservas
- Gestión de órdenes
- Panel administrativo
- Integración con MySQL

---

## Tecnologías

- Node.js
- Express
- MySQL
- JWT
- Render

---

## Frontend

https://spot-organico.vercel.app

---

## Estado

Backend funcional desplegado en producción.
